# 鬼リピ 〜連立方程式・代入法マスター〜

中学2年生で学習する「連立方程式」のうち、**代入法**で解くパターンを徹底的に練習できるドリルアプリです。
Chromebookなどの学校端末での動作に完全最適化しており、バグがなく快適に「鬼リピ（反復練習）」ができます。

---

## 🌟 アプリの特徴

1. **選べる2つのコース**
   * **基本コース**: 一方の式がすでに \(y = ax\) や \(y = ax + b\) などの形になっており、代入先が明確な問題（5問）
   * **応用コース**: 両方が \(y = \dots\) の形になっていたり、代入するために簡単な式変形（移項など）が必要な問題（5問）
2. **中学生が使いやすいダブル入力欄（案A）**
   * \(x\) の値と \(y\) の値を別々に入力できるため、入力のストレスがなく、計算だけに集中できます。
   * 分数やマイナスの入力に対応する専用の入力補助ボタンも搭載しています。
3. **Chromebook対応のIMEバグ防止**
   * 全角日本語入力のまま解答欄に入力してしまっても、自動的に半角英数字や正しい記号へマッピングされ、判定エラーを防ぎます。
4. **やる気を引き出す合格演出と学習履歴**
   * 8割以上正解（5問中4問以上）で合格💮となり、画面いっぱいにカラフルな紙吹雪が舞い散る楽しい演出。
   * 最近の学習履歴が20件まで記録され、いつでも自分の成長を確認できます。
5. **段階的ヒントと丁寧なステップ解説**
   * 答えに直結する4段階のヒントシステム（答えを見る前には「⚠️次は答えだよ！」と親切に警告）。
   * 間違えたときには、代入から答えを導くまでの途中計算プロセスがすべてステップバイステップで美しく表示されます。
6. **エンドレス復習モード**
   * 間違えた問題だけを抽出し、全問正解するまで永遠に解き直せる「復習モード」を搭載しています。

---

## 📂 ファイル構成

* **`index.html`**
  * 画面構造の定義。MathJax（数式表示）とMathLive（記述入力）の競合を完全に回避する順序でライブラリをロードします。
* **`style.css`**
  * Chromebook画面（1366x768）で全体スクロールが発生しないレイアウト。ユニバーサルデザインフォント（BIZ UDPGothic）を適用し、目に優しいダークテーマでスタイリングしています。
* **`main.js`**
  * コース管理、半角強制、答え合わせ、ヒント制御、学習履歴、復習モードなどのすべてのアプリ動作プログラム。
* **`questions.js`**
  * 整数や簡単な分数になるよう設計された逆算アルゴリズム搭載の問題自動生成エンジン。基本・応用各8種類のテンプレートを内蔵。
